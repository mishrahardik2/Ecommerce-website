# Ecommerce-website
This project is a responsive e-commerce clothing website developed using HTML, CSS, and JavaScript. It focuses on creating a clean, modern, and user-friendly shopping interface that allows users to browse clothing products and interact with dynamic website features without using any backend or frameworks.

The website includes a well-structured homepage showcasing multiple clothing items with images, prices, and brief descriptions. JavaScript is used to handle interactivity such as adding products to the cart and updating the cart state dynamically. CSS ensures a visually appealing layout with responsiveness across different screen sizes, making the website accessible on mobile, tablet, and desktop devices..  

Features

Fully responsive design for all screen sizes

Clothing product listing with images and prices

Add-to-cart functionality using JavaScript

Dynamic updates without page reload

Clean and modern UI design

Lightweight and fast performance
